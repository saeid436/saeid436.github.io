About Me

I work in machine learning and data science, with an academic background in electronic engineering and a master's in data science. My experience includes image processing, pattern recognition, and deep learning, with a focus on healthcare and computer vision applications. I've developed models for skin cancer detection, smart traffic systems, and NLP-based tools.

I started out working on low-level signal processing and embedded systems, including projects in C++ and assembly. Later, I transitioned into machine learning, focusing on applied deep learning using Python, PyTorch, and TensorFlow. Alongside model development, I’ve been working on performance optimization using CUDA and OpenMP for parallel computing and GPU acceleration.

I'm especially interested in projects where both accuracy and system performance matter — areas where research meets deployment, and models must scale and run efficiently in real-world environments.
